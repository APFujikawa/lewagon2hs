Tu web en 1,5 horas

Aprendé HTML y CSS con Le Wagon

1. Preparemos nuestro lugar de trabajo

  1. Abramos Sublime Text. Aparecerá una nueva pantalla negra.
  2. Crea una carpeta web en tu Escritorio. Selecciona y suelta ésta carpeta en la ventana del editor de texto Sublime Text.
  3. Sobre la parte izquierda (barra de navegación) de Sublime Text:
    • Botón derecho "Nuevo archivo"
    • Guardar como index.html con Cmd + S (Mac) o Ctrl + S (Windows)
    • Realizamos la misma acción para crear el archivostyle.css
    • Luego "Nueva carpeta" y creamos una carpeta llamadaimages
  4. Finalmente, doble click en index.html para abrirla con Chrome

2. Agregar contenido HTML

  • Para ahorrar tiempo, simplemente botón derecho en cada imagen y la guardamos en nuestra carpeta images
  • Luego, copiemos y peguemos este código de HTML en index.html
<html>
  <head>
    <meta charset="utf-8">
    <title>Nuestra propia web</title>
  </head>
  <body>
    <h1>Nuestra increíble web</h1>
    <p>Ésta es la línea importante que todos leen...</p>
    <p>
      <a href="#">Empezar!</a>
    </p>

    <img src="images/briefcase.png" alt="picture description" width="100">
    <h2>Rápido</h2>
    <p>Un servicio rápido, <strong>muy rápido</strong></p>

    <img src="images/diamond.png" alt="picture description" width="100">
    <h2>Simple</h2>
    <p>Un servicio simple, <strong>muy simple</strong></p>

    <img src="images/heart.png" alt="picture description" width="100">
    <h2>Increíble</h2>
    <p>Un servicio increíble, <strong>muy increíble</strong></p>

    <img src="images/laptop.png" alt="picture description" width="100">
    <h2>Interesante</h2>
    <p>Un servicio interesante, <strong>muy interesante</strong></p>

    <p>Estamos usando un gran template de ©Le Wagon</p>
  </body>
</html>

Extra - encontrá tus iconos personales

  • Podés seleccionar iconos en IconStore, The Noun Project or NucleoApp (tienen paquetes gratis).
  • Recomendación: también mantenete actualizado en plataformas para iconos con Product Hunt

3. CSS para fuentes y colores

Conectemos nuestro archivo de estilo style.css en la sección head del archivo de HTML

<head>
  <link href='style.css' rel='stylesheet'>
</head>

Luego copiamos y pegamos el siguiente código CSS en style.css y ésto hará que nuestra página tenga diseño!

  • Para fuentes, utilizamos Google Fonts
  • Para colores, utilizamos Colorzilla Chrome plugin para seleccionar colores de otras páginas web
  • A su vez, podemos encontrar colores en Coolors y en Color Hunt
/* style.css */
body{
  background: rgb(245,245,245);
  color: green;
  font-size: 25px;
  font-family: Helvetica;
}
h1{
  font-family: Courier;
  color: red;
  font-weight: lighter;
  font-size: 10px;
}
h2 {
  font-family: Courier;
  color: pink;
  font-weight: lighter;
  font-size: 8px;
}
a {
  color: yellow;
}
a:hover {
  text-decoration: none;
  color: purple;
}

5. Nombremos nuestras etiquetas

  • Agreguemos un identificador a nuestro 'banner' y 'footer' con id cómo <div id="banner"> y <div id="footer">
  • Nombremos las características de nuestra App con class cómo <div class="card">
  • Luego, agreguemos este código CSS
#banner{
  text-align: center;
  background-image: url("https://unsplash.it/1300/600?random");
  background-size: cover;
  padding: 150px;
  color: white;
  text-shadow: 1px 1px 5px black;
}
#banner h1 {
  color: white;
  font-size: 50px;
}
#banner p {
  color: white;
  opacity: 0.8;
  font-size: 30px;
  font-weight: lighter;
}
#footer{
  padding: 30px;
  background: rgb(30, 30, 30);
  color: lightgrey;
}
.card{
  padding: 50px;
  font-weight: lighter;
}

7. Bootstrap setup

Conectemos la 'stylesheet' de Bootstrap con nuestro proyecto en la sección head de nuestro HTML. Importante: antes de nuestra hoja de diseño 'stlye.css'

<head>
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">
  <link href='style.css' rel='stylesheet'>
</head>

Practiquemos cambiando la clase a class="btn btn-primary" y class="list-inline", en nuestra primera clase de Bootstrap
